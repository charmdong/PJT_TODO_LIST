# TO-DO LIST 만들기
-----
## 기간: 2019.07.08 ~ 2019.07.15

## 기술 스택
### 프론트엔드
1. HTML5
2. CSS
3. Javascript

### 백엔드
1. Java
2. Tomcat
3. MySql

## 요구 사항
1. TODO LIST 목록 조회 화면, TODO 등록 화면 개발
2. 순수 자바스크립트를 이용해 DOM, event 처리
3. maven 프로젝트로 생성
4. Redirect, forwarding 적용
5. ajax를 사용한 비동기 통신

## 주요 기능
1. 할 일 목록 조회
2. 할 일 상태 변경 (TODO -> DOING, DOING -> DONE)
3. 할 일 등록 (제목, 담당자, 우선순위 설정)
